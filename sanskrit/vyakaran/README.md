## Purpose
The primary objective of this project is to make the text available in its original form, free from the influence of subsequent commentators, allowing readers to engage directly with the author's thoughts. A secondary goal is to produce a high-quality PDF with an underlying Unicode layer. The extent to which these objectives have been fully realized remains open to evaluation.

## Local Compilation
The (La)TeX source files included in this repository can be compiled locally using engines such as XeLaTeX. This allows for the generation of the PDF documents directly on a local system without relying on cloud-based editors like Overleaf.

## Note on GRETIL Data
The `ot` folder contains a downloaded and converted Mahabhashya text from GRETIL. This approach was abandoned in favor of the resources listed above because the GRETIL transliteration removes original sandhi and isolates the text into individual padas. This fragmentation compromises the integrity of the original text for readers who are comfortable with continuous Devanagari and Sanskrit.

## Future Work
Ongoing work is required for numerical rendering and minor text fixes. These corrections can be implemented incrementally as one studies the Mahabhashya over time. 

As far as prakriyā goes, future work includes attempting to typeset the Pauṣpī Prakriyā in LaTeX for printing and study. The foundational files for this future work are contained exclusively in the `ashtadhyayi.zip` archive located on the main page of this repository. This zip file will contain strictly the work pertaining to Mataji and her prakriyās. For other related resources, scripts, and external projects, you can look into the `ot` directory.
